# Ultra-Light WooCommerce Size Chart 📏

A high-end, zero-bloat PHP snippet engineered to add custom size charts to WooCommerce products without compromising speed. 

Most size chart plugins are heavy and degrade your store's performance. I built this superior solution with a technical soul—keeping your WooCommerce environment fast, clean, and highly optimized.

## ✨ Technical Superiority
* **Zero Bloat:** No unnecessary CSS or JavaScript files loading on every page.
* **Lightning Fast:** Pure PHP implementation optimized for maximum WooCommerce performance.
* **Fully Customizable:** Easy to tweak dimensions and layout to fit your specific theme's architecture.

## 🚀 How to Implement
1. Copy the code from `ultra-light-size-chart.php`.
2. Paste it into your child theme's `functions.php` file or use a snippet manager plugin (like Code Snippets).
3. Customize the chart values directly in the array to match your product dimensions.
